CoursedRepository
=================
System for punishing students.

Frameworks and Tools
-----------------------------------
* Java&IDE: JDK8, IntelliJ IDEA
* Backend:  Spring, REST, SpringMVC, Spring Security, JPA(Hibernate) 
* Frontend: JavaScript, JQuery, Thymeleaf, Bootstrap
* Database: MySQL
* Web Server: Tomcat
* Build Tool: Maven
* Other: Logback
